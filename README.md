🧠 Unsaid – AI Mental Health Companion

🔗 Live Demo: https://v0-unsaid-ai-mental-health-companion.vercel.app/

🌟 Overview

Unsaid is an AI-powered mental health companion that provides a safe and private space for users to express their thoughts and emotions.

It helps users reflect, journal, and have supportive conversations anytime they need.

🚀 Features
💬 AI chat for emotional support
📓 Journaling and self-reflection
🧘 Wellness activities
🔒 Privacy-focused experience
⚡ Available anytime
🛠️ Tech Stack
Frontend: React / Vite / Tailwind CSS
Backend: Node.js (if used)
AI: LLM APIs
Deployment: Vercel
⚙️ Installation
# Clone the repository
git clone https://github.com/your-username/unsaid-ai.git

# Go to project folder
cd unsaid-ai

# Install dependencies
npm install

# Run the app
npm run dev
🌐 Usage
Open the app
Start chatting with the AI
Write your thoughts or journal
Explore wellness features
