## The Download Process (What Users Do)

### Step 1: Go to Voice Page
User navigates to Voice section of app

### Step 2: Click Download
User sees three models:
1. Speech Recognition (800 MB) - Understands voice
2. AI Response (1.4 GB) - Creates replies
3. Voice Output (1.1 GB) - Reads aloud

### Step 3: Download Each One
User clicks "Download Now" button for each model

### Step 4: Wait 15-30 Minutes
Progress bars fill up. User waits (can do other things).

### Step 5: See Green Checkmarks
All three models download successfully. User sees ✓

### Step 6: Done!
App says "Ready for Offline Use". Models stay on device forever.
